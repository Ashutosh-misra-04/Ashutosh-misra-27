## **Hybrid PSO-BO-LSTM Framework for Cloud Load Balancing and Host Utilization Prediction**

I completed a research project focused on **cloud load balancing and host utilization prediction** using a hybrid deep learning and optimization framework.

- 🔭 I completed the project **Hybrid PSO-BO-LSTM Framework for Cloud Load Balancing and Host Utilization Prediction**

- ☁️ The project addresses the challenge of **efficient task scheduling and resource allocation in cloud computing environments**

- 🤖 The framework combines **Long Short-Term Memory (LSTM)** with two optimization techniques:
  - Particle Swarm Optimization (PSO)
  - Bayesian Optimization (BO)

- ⚙️ PSO performs **global exploration of hyperparameters**, while BO refines the best solutions through **probabilistic optimization**, improving model convergence and accuracy.

- 📊 A **synthetic dataset of 1000 cloud tasks** was generated containing features such as:
  - Task execution time
  - CPU cores
  - Memory requirements
  - Network bandwidth
  - Priority level
  - Virtual machine ID

- 📈 The LSTM model learns **temporal patterns in task workloads** to predict future execution time and improve resource scheduling.

- 📉 Model performance was evaluated using **Mean Squared Error (MSE) and Mean Absolute Error (MAE)**.

- 🚀 The hybrid **PSO-BO-LSTM model achieved better performance than standalone PSO-LSTM and BO-LSTM models**, reducing prediction error and improving convergence speed.

- 💻 Technologies used:

  - TensorFlow
  - Optuna
  - NumPy

  - Data visualization tools

- 🌱 This work demonstrates how **combining metaheuristic optimization with deep learning can improve predictive modeling and resource management in cloud systems**
